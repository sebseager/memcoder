# Artifacts

This directory stores generated MemCoder artifacts.

The initial pilot layout is:

```text
artifacts/
  antirez__kilo/
    easy/
      docs/
      qas/
      loras/
      ledger.md
      lora_store.json
      eval_results.jsonl
```

Generated artifacts should record the target repository identity:

```text
repo_id: antirez__kilo
path: target_repos/antirez__kilo
commit: 323d93b29bd89a2cb446de90c4ed4fea1764176e
```
