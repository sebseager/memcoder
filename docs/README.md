# Docs

This directory holds project-level documentation that is not specific to one
source package or artifact directory.

- `initial_plan.md`: original MemCoder project plan, moved from the repository
  root.
- `todo.md`: original task checklist, moved from the repository root.
- `observations.md`: pilot evaluation observations that previously lived in the
  root README.
- `evaluation.md`: replication guide for the included evaluation.
- `ai_usage.md`: disclosure of AI tools used during development.
